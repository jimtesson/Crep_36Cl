Start computing ages

  • CREp (Cosmic Ray Exposure program) is a new MatlabÂ©/Octave online tool to compute exposure ages using 10Be and 3He.
  • This program includes both the Lal/Stone time dependent scaling model (Lal, 1991 ; Stone, 2000), and the LSD model (Lifton et al., 2014).
    Users can choose between the standard atmosphere pressure, or the ERA40 atmosphere model.
    The time dependent correction may be done using several different Virtual Dipole Moment databases.
  • CREp is associated with a new online database, ICE-D, that includes all the existing empirical production rates calibrations.
  • Users can either select global, regional or local production rates to compute exposure ages.

For exhaustive description of calculation: Martin et al., 2017
